# **Card Game — Addressables CDN**

This repository serves as the **CDN for Addressables bundles** used by the **Card Game (WebGL)** project.
All files are hosted via GitHub Pages and available at:

**[https://spikethehedgehogdev.github.io/card-game-bundles/](https://spikethehedgehogdev.github.io/card-game-bundles/)**

## **Structure**

* `catalog_0.1.0.json` — main Addressables catalog
* `catalog_0.1.0.hash` — catalog hash
* `*.bundle` — asset bundles
